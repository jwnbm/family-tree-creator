# Screenshots

このディレクトリにはアプリケーションのスクリーンショットを保存してください。

## 必要なスクリーンショット

以下のスクリーンショットを撮影してこのディレクトリに保存してください：

### 1. main-view.png
- アプリケーションのメインキャンバス画面
- サンプルデータを読み込んだ状態
- 4世代の家系図が表示されている様子
- 推奨サイズ: 1200x800px 以上

**撮影手順:**
1. アプリケーションを起動
2. 「サンプル」ボタンをクリック
3. キャンバス全体が見えるようにズーム調整
4. スクリーンショットを撮影

### 2. person-editor.png
- 個人エディタの画面
- 関係管理セクションが表示されている状態
- 推奨サイズ: 600x800px 以上

**撮影手順:**
1. サンプルデータから任意の人物を選択
2. 左側の「個人」タブの詳細が表示されている状態
3. 関係セクションまでスクロールして表示
4. スクリーンショットを撮影

### 3. family-groups.png
- 家族グループ機能の画面
- 色分けされた家族グループが表示されている状態
- 推奨サイズ: 1200x800px 以上

**撮影手順:**
1. サンプルデータを読み込んだ状態
2. キャンバス上で家族グループの背景色が見える位置に調整
3. 複数のグループが見えるようにズーム調整
4. スクリーンショットを撮影

## 画像形式

- フォーマット: PNG (推奨) または JPG
- 背景: アプリケーションの実際の表示
- 品質: 高品質（圧縮しすぎない）

## 保存方法

Windows:
```powershell
# スクリーンショットを撮影後、このディレクトリに保存
# ファイル名を上記の通りにリネーム
```

画像編集ソフトを使用して、必要に応じてトリミングやリサイズを行ってください。
